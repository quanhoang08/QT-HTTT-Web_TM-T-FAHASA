# Fahasa_Figma (Frontend)

## Chạy dự án

1. Cài phụ thuộc: `npm i` (hoặc `pnpm i` nếu bạn dùng pnpm)
2. Giao diện dev: `npm run dev` (Vite, thường http://localhost:5173)
3. Biến môi trường: có thể đặt `VITE_API_URL` (mặc định: `http://localhost:8080/api`) trỏ tới backend bạn đang chạy

**Trang quản trị (Admin):** cần đăng nhập tài khoản có `role: admin` trên server. Tài khoản mẫu và bước seed — xem mục [Hướng dẫn Admin](../README.md#hướng-dẫn-admin) trong README thư mục gốc.
